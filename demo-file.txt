FastShare is designed to be the fastest file sharing tool for local networks, built with Rust for maximum performance and reliability.

Key features include:
- Ultra-fast TCP-based transfers optimized for speed
- Real-time progress tracking with beautiful progress bars
- Cross-platform compatibility (Windows, macOS, Linux)
- Simple and intuitive command-line interface
- Automatic network detection and setup
- Human-readable file size formatting
- Robust error handling and recovery

Perfect use cases:
- Sharing large documents between devices
- Transferring photos and videos quickly
- Moving files between development machines
- Sharing presentations before meetings
- Quick file backups across devices

Technical details:
- Uses 64KB chunks for optimal throughput
- Implements efficient async I/O with Tokio
- Minimal protocol overhead for maximum speed
- Direct TCP streaming without unnecessary compression
- Progress tracking with minimal performance impact
